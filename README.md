## Description
This script cleans and prepares three datasets for analysis of how call 
volumes changed when CAHOOTS stopped services in Eugene in April 2025.

## Dependencies
R 
- tidyverse
- readxl
- lubridate

## Data
Place the following files in the same directory as the script:
- Eugene_CAD_data_noloc/ (folder with 11 .csv files)
- 2015-2025 SPD Calls for Service.xlsx
- MCSLC.xlsx

## How to Run
Open Week-5.Rmd in RStudio and click "Knit"
